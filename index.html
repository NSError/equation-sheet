About

This equation sheet is built by Chris Miller. It was originally constructed using a large quantity of $\LaTeX$ as well as a few hand-written PostScript diagrams in 2009. It is in the process of being re-translated to use more modern HTML5. Key technologies include:

  • MathJax: A nifty technology that uses JavaScript to render embedded $\TeX$ to HTML/CSS.
  • HTML5 Canvas (for the pretty diagrams).
  • 960gs: An über CSS layout.
  • TextMate: Still the best text editor I can find.

The original product was, as mentioned, written in $\LaTeX$. It used a lot of custom style templates, such as one which got me the two-column effect. I also hand-wrote the diagrams in PostScript. The original files are here:

  • eqnsheet.pdf PDF version of the original, as run through my big massive $\LaTeX$ stack.
  • eqnsheet.tex The $\LaTeX$ source.
  • trig_diagram1.tex Some hand-crafted PostScript to create a lovely diagram showing the trigonometric identities.
  • unit_circle.tex Some hand-crafted PostScript to create a lovely unit-circle.
$$ \sin \theta = \frac{y}{r} $$ $$ \cos \theta = \frac{x}{r} $$ $$ \tan \theta = \frac{y}{x} $$

Reciporicals

$ \cot \theta = \frac{1}{\tan \theta} $
$ \sec \theta = \frac{1}{\cos \theta} $
$ \csc \theta = \frac{1}{\sin \theta} $

Definitions

$ \cot \theta = \frac{ \cos \theta }{ \sin \theta } $
$ \sec \theta = \frac{ 1 }{ \cos \theta } $
$ \csc \theta = \frac{ 1 }{ \sin \theta } $

Hyperbolic

$ \sinh \theta = \frac{ e ^{\theta} - e ^{-\theta} }{ 2 } $
$ \cosh \theta = \frac{ e ^{\theta} + e ^{-\theta} }{ 2 } $
$ \tanh \theta = \frac{ e ^{\theta} - e ^{-\theta} }{ e ^{\theta} + e ^{-\theta} } $

Pythagorean

$$ \sin ^{2} \theta + \cos ^{2} \theta = 1 $$
$$ \tan ^{2} \theta + 1 = \sec ^{2} \theta $$
$$ 1 + \cot ^{2} \theta = \csc ^{2} \theta $$

Cofunction

$ \sin (\frac{\pi}{2} - \theta ) = \cos \theta $
$ \cos (\frac{\pi}{2} - \theta ) = \sin \theta $
$ \tan (\frac{\pi}{2} - \theta ) = \cot \theta $

Even/Odd

$ \sin -\theta = -\sin \theta $
$ \cos -\theta = \cos \theta $
$ \tan -\theta = -\tan \theta $

Double Angle

$ \sin 2\theta = 2\sin\theta\cos\theta $
$ \cos 2\theta = \cos ^{2} \theta - \sin ^{2} \theta $
$ \cos 2\theta = 1 - 2\sin ^{2}\theta $

Half-Angle

$ \sin ^{2} \theta = \frac{1 - \cos 2 \theta}{2} $
$ \cos ^{2} \theta = \frac{1+\cos 2\theta}{2} $

Addition

$ \sin ( a + b ) = \sin a \cos b + \cos a \sin b $
$ \cos ( a + b ) = \cos a \cos b - \sin a \sin b $

Subtraction

$ \sin ( a - b ) = \sin a\cos b - \cos a\sin b $
$ \cos ( a - b ) = \cos a\cos b + \sin a\sin b $

Sum

$ \sin u + \sin v = 2\sin \frac{u+v}{2}\cos \frac{u-v}{2} $
$ \cos u + \cos v = 2\cos \frac{u+v}{2}\cos \frac{u-v}{2} $

Product

$ \sin u\sin v = \frac{1}{2}\left[ \cos(u-v)-\cos(u+v) \right] $
$ \cos u\cos v = \frac{1}{2}\left[ \cos(u-v)+\cos(u+v) \right] $
$ \sin u\cos v = \frac{1}{2}\left[ \sin(u+v)+\sin(u-v) \right] $
$ \cos u\sin v = \frac{1}{2}\left[ \sin(u+v)-\sin(u-v) \right] $

Derivative Formulas

General Rules

$ \begin{array}{l} \frac{d}{dx} \left[ f(x) \pm g(x) \right] = f ^\prime (x) \pm g ^\prime (x) \\ \frac{d}{dx} \left[ f \left( g(x) \right) \right] = f ^\prime \left( g(x) \right) g ^\prime (x) \\ \frac{d}{dx} \left[ cf(x) \right] = cf ^\prime (x) \\ \frac{d}{dx} \left[ f(x)g(x) \right] = f ^\prime (x)g(x) + f(x)g ^\prime (x) \\ \frac{d}{dx} \left[ \frac{f(x)}{g(x)} \right] = \frac{f ^\prime (x)g(x)-f(x)g ^\prime (x)}{ \left[ g(x) \right] ^{2}} \end{array} $

Power Rules

$\frac{d}{dx}x^{r}=rx^{r-1}$
$\frac{d}{dx}c=0$
$\frac{d}{dx}cx=c$
$\frac{d}{dx}\sqrt{x}=\frac{1}{2\sqrt{x}}$

Exponential Rules

$ \begin{array}{ll} \frac{d}{dx}\: e^{x} = e^{x} \\ \frac{d}{dx}\: e^{u(x)} = e^{u(x)}u ^\prime (x) \end{array} $
$ \begin{array}{ll} \frac{d}{dx}\: a^{x} = a^{x} \ln a \\ \frac{d}{dx}\: e^{rx} = re^{rx} \end{array} $
$ \frac{d}{dx} \ln x = \frac{1}{x} $

Inverse Trigonometric Rules

$ \begin{array}{ll} \frac{d}{dx}\: \sin ^{-1} x = \frac{1}{\sqrt{1 - x ^{2} }} \\ \frac{d}{dx}\: \tan ^{-1} x = \frac{1}{1 + x ^ {2}} \\ \frac{d}{dx}\: \sec ^{-1} x = \frac{1}{|x|\sqrt{x^{2}-1}} \end{array} $
$ \begin{array}{ll} \frac{d}{dx}\: \cos ^{-1} x = \frac{-1}{\sqrt{1 - x ^{2} }} \\ \frac{d}{dx}\: \cot ^{-1} x = \frac{-1}{1 + x ^ {2}} \\ \frac{d}{dx}\: \csc ^{-1} x = \frac{-1}{|x|\sqrt{x^{2}-1}} \end{array} $

Hyperbolic Rules

$ \newcommand{\sech}{\text{sech}} \newcommand{\csch}{\text{csch}} \begin{array}{ll} \frac{d}{dx}\: \sinh x = \cosh x \\ \frac{d}{dx}\: \tanh x = \sech x \\ \frac{d}{dx}\: \sech x = -\sech x \tanh x \end{array} $
$ \newcommand{\sech}{\text{sech}} \newcommand{\csch}{\text{csch}} \begin{array}{ll} \frac{d}{dx}\: \cosh x = \sinh x \\ \frac{d}{dx}\: \coth x = -\csch ^{2} x \\ \frac{d}{dx}\: \csch x = -\csch x \coth x \end{array} $

Inverse Hyperbolic Rules

$ \begin{array}{ll} \frac{d}{dx}\: \sinh ^{-1} x = \frac{1}{\sqrt{1 + x ^{2}}} \\ \frac{d}{dx}\: \tanh ^{-1} x = \frac{1}{1 - x ^{2}} \\ \frac{d}{dx}\: \sech ^{-1} x = \frac{-1}{x \sqrt{1 - x ^{2}}} \end{array} $
$ \begin{array}{ll} \frac{d}{dx}\: \cosh ^{-1} x = \frac{1}{\sqrt{x ^{2} - 1}} \\ \frac{d}{dx}\: \coth ^{-1} x = \frac{1}{1 - x ^{2}} \\ \frac{d}{dx}\: \csch ^{-1} x = \frac{-1}{|x|\sqrt{x ^{2} + 1}} \end{array} $

Trigonometric Rules

$ \begin{array}{ll} \frac{d}{dx}\: \sin x = \cos x \\ \frac{d}{dx}\: \tan x = \sec ^{2} x \\ \frac{d}{dx}\: \sec x = \sec x \tan x \end{array} $
$ \begin{array}{ll} \frac{d}{dx}\: \cos x = -\sin x \\ \frac{d}{dx}\: \cot x = -\csc ^{2} x \\ \frac{d}{dx}\: \csc x = -\csc x \cot x \end{array} $
